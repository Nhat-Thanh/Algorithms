</style><svg style="display: none;"><defs id="MathJax_SVG_glyphs"></defs></svg><p>Preprocessor directives are lines included in the code preceded by a hash sign (#). These lines are directives for the preprocessor. The preprocessor examines the code before actual compilation of code begins and resolves all these directives before any code is actually generated by regular statements.</p>

<pre><code>#define INF 10000000
if( val == INF) {
//Do something
}
After the preprocessor has replaced the directives, the code will be
if( val == 10000000) { //Here INF is replaced by the value with which it's defined.
//Do something
}
</code></pre>

<p>You can also define function macros which have parameters.</p>

<pre><code>#define add(a, b) a + b
int x = add(a, b);

The second statement after the preprocessor has replaced the directives will be:
int x = a + b;
</code></pre>

<p>To know more about preprocessor directives, you can go to this <a href="http://www.cplusplus.com/doc/tutorial/preprocessor/">link</a></p>
